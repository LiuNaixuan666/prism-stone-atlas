# 版本更新记录

## 0.4.0 - 2026-07-21

- 将 2073 张已下载的图鉴图片打包进 PWA，不再依赖 Fandom 图片服务器。
- 安装或更新 PWA 时预缓存完整图鉴图片，断网后也能继续查看。
- 新增本地图包同步脚本，并校验图鉴记录和图片文件一一对应。
- 保留 2 条原始资料缺图的记录，继续显示“图片暂缺”。

## 0.3.0 - 2026-07-21

- 新增完全静态的 GitHub Pages 本机版，与现有网站并行发布。
- GitHub Pages 版本不显示登录或云端功能，只保留本机数据、恢复点和导入导出。
- 调整图鉴数据、安装清单和离线程序，使其同时支持根域名和仓库子路径。
- 新增 GitHub Actions，合并到 `main` 后自动构建并更新 Pages。
- 保留现有 Sites 版本及其可选云端备份能力。

## 0.2.0 - 2026-07-21

- 将收藏改为本机优先，登录后不再自动上传或自动覆盖数据。
- 将云端功能拆分为手动“备份到云端”和“从云端恢复”。
- 云端恢复采用安全合并，并在操作前创建本机恢复点。
- 加入最多 10 份滚动本机恢复点，可撤回导入、恢复或误清空。
- 云端写入加入版本冲突保护，避免旧设备覆盖较新的备份。
- 加强 PWA 核心程序与图鉴目录的离线缓存和更新流程。
- 用数据安全与离线能力测试替换失效的初始模板测试。

## 0.1.0 - 2026-07-21

- 建立手机优先的 Prism Stone 收藏图鉴。
- 支持图鉴搜索、分类筛选、持有状态、心愿、数量和备注。
- 加入统计、缺少清单和自定义棱石。
- 使用 IndexedDB 与 localStorage 保存当前设备数据。
- 加入收藏备份导出、导入和可选账号云端同步。
- 加入 PWA 安装入口与基础离线缓存。
- 公开项目源码，并确定后续采用“本地优先、云端可选”的维护方向。
